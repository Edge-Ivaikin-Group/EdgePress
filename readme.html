Welcome to EdgePress

EdgePress is a lean, fast alternative to WordPress, designed to meet the needs of developers and businesses who value simplicity, control, and modern standards. With EdgePress, you get enhanced security, developer autonomy, and a CMS built for the future.

Installation: 5-Minute Setup

  1. Download EdgePress and unzip it in a new directory.
  2. Open wp-admin/install.php in your browser and follow the prompts to set up the configuration file.
  3. If automatic setup fails, manually edit wp-config-sample.php and save as wp-config.php.
  4. Once set up, the installer will configure the database and create the necessary tables.
  5. Log in using your chosen username and password.

System Requirements

  • PHP version 7.4 or greater.
  • MySQL version 5.6 or greater, or MariaDB version 10.1 or greater.
  • mod_rewrite Apache module.
  • HTTPS support recommended.
